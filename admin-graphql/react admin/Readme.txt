## Inappropriate Review Scenarios

1. **Offensive Language**: Reviews containing offensive or inappropriate language that violates community guidelines.
2. **Spam or Advertising**: Reviews that are spam or contain unauthorized advertisements.
3. **Irrelevant Content**: Reviews that are irrelevant to the product or contain misleading information.

## Implementation


- The frontend `ReviewManagement` component allows admins to delete reviews and displays the deletion message.
